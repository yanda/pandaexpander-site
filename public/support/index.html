Let’s get it working.

Most setup issues take one permission check or snapshot refresh. If not, the app can prepare a privacy-safe support report.

Mac expansion isn’t firing

  1. Open Panda Expander and confirm the lower-left status says Expansion ready.
  2. If access is required, choose Allow, enable Panda Expander in System Settings → Privacy & Security → Accessibility, then return to the app.
  3. Confirm expansion is not paused from the menu-bar panda.
  4. Open Getting Started and type ;hello in its test field after adding examples.
  5. Try the same abbreviation in TextEdit. Secure fields and password managers intentionally never expand.

The iOS keyboard or snippets are missing

  1. Open the Panda Expander app and wait for the Library to finish iCloud sync.
  2. Open the Keyboard tab and tap Refresh.
  3. Confirm the status shows compatible snippets and a recent update time.
  4. In Settings → General → Keyboard → Keyboards, remove and re-add Panda Expander if it no longer appears.
  5. Do not enable Full Access. The keyboard does not need it.

Send useful feedback safely

On Mac, choose Panda Expander → Settings → Health → Copy Support Report, or choose Send Feedback… from the Help menu. The report omits snippet names, abbreviations, snippet content, typed text, form values, clipboard contents, and surrounding text.

On iPhone or iPad, include the app version from TestFlight, the app where the problem occurred, and whether the Keyboard page showed a current snapshot.
