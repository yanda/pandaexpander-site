# Panda Expander website

Static website for [pandaexpander.com](https://pandaexpander.com), deployed
through Cloudflare Pages.

## Local preview

```sh
npm run serve
```

## Build

```sh
npm run build
```

Cloudflare Pages uses `npm run build` with `dist` as the output directory.
The site has no accounts, cookies, analytics, or backend.
